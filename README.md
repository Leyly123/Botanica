# 🌿 Tu Botánica

Sitio web desarrollado para la gestión, visualización y difusión de contenido relacionado con plantas, botánica y oportunidades laborales dentro del sector.

---

## 📂 Estructura del Proyecto

```text
Tu botanica/
│
├── Css/
│   └── Archivos de estilos CSS
│
├── img/
│   └── Imágenes, iconos y recursos gráficos
│
├── index.html
├── login.html
├── registrarse.html
├── plantas.html
├── galeria.html
├── vacantes.html
└── vacantes_blog.html
```

---

## 📄 Descripción de Archivos

### 🏠 index.html
Página principal del sitio.

**Contiene:**
- Presentación del proyecto.
- Información general.
- Navegación hacia las demás secciones.

---

### 🔐 login.html
Página de inicio de sesión para usuarios registrados.

**Funcionalidades:**
- Formulario de acceso.
- Validación de credenciales.
- Redirección a secciones privadas.

---

### 📝 registrarse.html
Página de registro de nuevos usuarios.

**Incluye:**
- Formulario de creación de cuenta.
- Captura de datos básicos.
- Validaciones de entrada.

---

### 🌱 plantas.html
Catálogo o listado de plantas disponibles.

**Características:**
- Información descriptiva.
- Organización de especies.
- Consulta de contenido botánico.

---

### 🖼️ galeria.html
Galería visual del proyecto.

**Permite:**
- Visualizar imágenes de plantas.
- Explorar contenido multimedia.
- Mejorar la experiencia visual del usuario.

---

### 💼 vacantes.html
Sección dedicada a oportunidades laborales.

**Incluye:**
- Publicación de vacantes.
- Información de cargos disponibles.
- Consulta de ofertas de empleo.

---

### 📚 vacantes_blog.html
Blog o sección informativa relacionada con las vacantes.

**Contiene:**
- Artículos informativos.
- Noticias y actualizaciones.
- Consejos para postulantes.

---

## 🎯 Objetivo del Proyecto

Desarrollar una plataforma web intuitiva enfocada en la divulgación botánica, visualización de especies vegetales y publicación de oportunidades laborales relacionadas con el sector ambiental y agrícola.
